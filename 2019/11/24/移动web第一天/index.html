移动web第一天

什么是HTML5？

不是一门新的语言，而是html第五次重大修改

html5修改了什么内容？

1.新增了语义化标签 
2.新增了音频  视频  canvas画图
3.新增input标签  新增标签表单的属性
4.新增本地存储  地理定位 
5.新增对类名操作 
6.新增自定义属性的设置

语义化标签：（适用于移动端）

1.header 头部标签
2.nav  导航标签
3.article 内容标签
4.section 相当于一块区域
5.aside 侧边栏
6.footer 尾部 
7.main 主体标签

音频

标签： audio
属性：
    src： 路径
    controls：控制器
    autoplay:自动播放  
    loop:循环播放

视频：

​ 标签 video
​ 属性：
​ src： 路径
​ controls：控制器
​ autoplay:自动播放
​ loop:循环播放
​ poster：默认播放前的第一帧。
​ muted：静音播放
​ width与height 视频的宽高
​ 等比例放大缩小（只需要设置一个即可）

​ 8.search： 人性化体验
​ 9.time：时间

新增属性

​ required： 当前项为必填、
​ placeholder：默认提示信息
​ disabled：禁用
​ readonly：只读（只允许读取 不允许修改）
​ autofocus：自动获取焦点。
​ autocomplete：自动提示信息 注意： 当前文本框必须拥有name属性 必须真正提交过 off（不提示）|on（提示）
​ multiple：多选（文件上传） 上传之后的图片存储在 files[0]

Css3新增：

​ 1.选择器
​ 2.旋转 移动 缩放 动画
​ 3.旋转 移动 缩放（3d）更加立体

选择器：

属性选择器
1.div[class] {
color: red;
}
2.[class] {
color: pink;
}
3.[class=”box”] {
color: red;
}
4. 以b开头选中
[class^=”b”] {
color: red;
}
5.以b结尾的选中
[class$=”b”] {
color: red;
}
6.只要包含b就会选中
[class=”b”] {
color: red;
}
*结构伪类选择器**
1).选择当前元素的父级元素的第一个li元素
1.元素:first-child
2.元素:last-child
3.元素:nth-child(n) -2+n 前两个孩子
2).选择当前元素的父级元素的同类型第一个li元素
1.元素：first-of-type
2.元素：last-of-type
3.元素：nth-of-type(n)
兄弟选择器
1).选择当前元素的后一个兄弟元素
元素+元素
2).选择当前元素的后面的所有兄弟元素
元素~元素
伪元素
元素::before 在元素里面的最前面添加内容
元素::after 在元素里面的最后面添加内容
注意：
1.表单元素无法添加伪元素
2.用伪元素添加之后是行内元素
3.必须有content;

扩展：
1.选中当前元素的第一行：
div::first-line {
color: red;
}
2.选中当前元素的第一个字：
div::first-letter {
color: red;
}
3.鼠标选中的状态：
div::selection {
color: red;
background: blue;
}
